¿Como hacer un repositorio en Github?

1. Primero en nuestra cuenta de Github seleccionamos New




2. Colocamos el nombre de nuestro repositorio, la descripcion es opcional y seleccionamos create repository




3. abrimos un visual Studio code con nuestro codigo, luego seleccionamos la opccion terminal




4. buscamos nuestra carpeta con el comando cd, en este caso esta en el escritorio




5. colocamos el nombre de nuestra carpeta




6. estos comandos son para guardar nuestras credenciales un nuestra computadora, si estamos en otra computadora cambiamos el --global por --local




7. colocamos git init, luego git add .(el punto es para seleccionar todo) y finalmente el comando git commit




8. vamos a nuestro respositorio en Github y copiamos el enlace




9. colocamos el comando git remote add origin seguido del link que copiamos




10. finalmente subimos nuestro archivo con el comando git push -u origin master




listo ya tenemos nuestro archivo en Github




Extras

git clone /path/to/repository

Crea una copia local del repositorio

git status

Lista un estado actual del repositorio con lista de archivos modificados o agregados

git pull

Para poder fusionar todos los cambios que se han hecho en el repositorio local trabajando.

git merge branch-name

Este comando se usa para fusionar una rama con otra rama activa:

git checkout -b bitlab

Crea una nueva rama llamada "bitlab" y cámbiate a ella usando

git checkout master

vuelve a la rama principal

git branch -d bitlab

borrar una rama
